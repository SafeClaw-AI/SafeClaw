# SafeClaw

[中文](#中文) | [English](#english)

## 中文

SafeClaw™ 是一个为每一个普通用户而生的安全执行系统。  
它希望让技术变得更简单、更透明、更值得信任。

我们相信：
- 工具应该让人轻松，而不是让人紧张
- 系统应该可控，而不是神秘
- 安全应该是默认，而不是奢侈
- 每个人都值得被尊重，不论是否是开发者

SafeClaw 还在早期阶段，我们正在探索最适合用户的方向。  
如果你愿意关注、提出想法或分享你的使用体验，我们会非常感激。

## English

SafeClaw™ is a secure execution system designed for everyone —  
not just developers, but anyone who wants technology to feel simple, transparent, and trustworthy.

We believe that:
- Tools should make life lighter, not heavier
- Systems should be understandable, not mysterious
- Security should be the default, not a luxury
- Every user deserves respect, regardless of technical background

SafeClaw is still in its early stage.  
Your thoughts, ideas, and experiences are warmly welcome.
