# SafeClaw V1 任务分级

## P0
必须优先完成，决定 V1 是否成立：

- 流程录制稳定性修复
- 键鼠回放稳定性修复（含 `dry-run` 与真实执行一致性）
- 本地任务执行稳定性修复（PowerShell / CMD / Bash）
- 自然语言任务预览升级为“风险 → 操作 → 确认”
- 底层不可修改的安全红线锁
- 安全控制默认全关，改为必须手动开启
- 一键取消可靠性与状态反馈
- 审计与全流程回放可读性
- 实时心跳状态条
- README 与产品文案统一到 V1 定位

## P1
V1 建议完成，用来强化体验与信任：

- Web Doctor 排障体验优化
- 原生龙虾灵魂对话语气统一
- 本地基础记忆系统（查看 / 删除 / 清空）
- 启动与离线启动路径打磨
- 历史前端问题与测试遗留问题清理

## Frozen
V1 冻结，不进入当前开发范围：

- OpenClaw 迁移兼容
- 人机协同、桌面流、Skill 体系
- 复杂流程编排、可视化编辑
- 灵魂自定义、预设角色库
- 高级参数配置
- 本地大模型适配
- 云端 API 对接
- 龙虾搜索

## 说明

本清单用于 V1 收口，不代表这些方向永远不做；统一在 V1 发布后，基于真实用户反馈再重新评估。
