# SafeClaw Vision

## 中文

SafeClaw 希望成为一个真正面向普通用户的安全执行系统。

它关注的不只是“能不能执行”，而是：
- 用户能不能看懂发生了什么
- 用户能不能知道风险在哪里
- 用户能不能在需要时停下来、确认、恢复或退出
- 用户能不能在使用自动化与 AI 时感到安心

SafeClaw 的核心愿景是：
- 让安全成为默认，而不是额外选项
- 让系统更透明，而不是更神秘
- 让体验更友好，而不是更有门槛
- 让技术更值得信任，而不是更难接近

我们希望 SafeClaw 最终不仅服务开发者，也服务学生、创作者、小团队与每一个普通用户。

## English

SafeClaw aims to become a secure execution system built for real users, not only technical experts.

It is not only about whether something can run, but also whether users can:
- understand what is happening
- see where the risks are
- pause, confirm, recover, or stop when needed
- feel calm and confident while using automation and AI

The core vision of SafeClaw is to:
- make security the default, not an extra
- make systems transparent, not mysterious
- make user experience friendly, not intimidating
- make technology more trustworthy and more approachable

In the long run, SafeClaw is meant to serve not only developers, but also students, creators, small teams, and everyday users.
